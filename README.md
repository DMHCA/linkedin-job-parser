# LinkedIn Job Parser

A simple Spring Boot application that parses job listings from LinkedIn.

## Features

- Parses job listings from LinkedIn
- Extracts job title, company, location and description
- Stores data in PostgreSQL
- Designed as a simple backend parser service

## Tech Stack

- Java 21
- Spring Boot
- Maven
- PostgreSQL
- Jsoup
